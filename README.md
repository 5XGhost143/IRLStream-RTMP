# IRL RTMP

**Use Docker...**

NODEJS IS REQUIRED!

1. Step: docker-compose build
2. Step: docker-compose up

Use:

Start it and copy your stream key!

Your Streamlabs for iOS or Android (OR ANYTHINH ELSE WITH CUSTOM RTMP) StreamKey: username?key=yourcopiedkey
